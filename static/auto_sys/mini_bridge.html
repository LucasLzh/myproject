Mini bridge is a Wi-Fi to RF+ interface. It builds a connection via user's Wi-Fi router at home and enables user to control motorized window covering and sunshading products with designated smartphone applications and interaction with ecosystem such as Amazon Alexa, Google Assistant,etc, via cloud API.
